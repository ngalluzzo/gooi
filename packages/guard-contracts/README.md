# @gooi/guard-contracts

Typed contracts for guard and invariant plans, evaluation envelopes, typed errors,
semantic judge ports, and guard violation signal payloads.

## Features

- Layered guard/invariant plan contracts by primitive kind
- Structural and semantic tier contract shapes
- Semantic confidence and sampling policy contracts
- Typed evaluation and violation envelope contracts
- Stable guard error taxonomy

- `@gooi/guard-contracts/plans/guard-plan`
