# @gooi/scenario-contracts

Typed contracts for persona and scenario plans, run envelopes, reports, and
scenario error taxonomy.

## Features

- Persona and scenario plan contracts
- Trigger/expect/capture step model contracts
- Deterministic generated-trigger lock snapshot contracts
- Typed scenario run and suite report contracts
- Stable scenario error codes

## Export Paths

- `@gooi/scenario-contracts/plans/scenario-plan`
- `@gooi/scenario-contracts/envelopes/scenario-envelopes`
- `@gooi/scenario-contracts/reports/scenario-reports`
- `@gooi/scenario-contracts/errors/scenario-errors`
